This plugin produces a simple copyright text in the footer which is editable. It also produces a copyright shortcode.
 
== Description ==
 
This plugin produces copyright text in the footer. It also produces a copyright shortcode, which you can display with the shortcode &#91;copyright&#93;.

Currently, you can edit the text, display, text alignment, padding, font size, font color, and bar color. You can find these edits under `Dashboard > Tools > Copyright`.

We open to adding new features so please let us know if you would like something more.

= Plugin Features =

* Free!
* Edit the text's content & size
* Show or hide your copyright footer
* Use of shortcode
* Set the text alignment
* Adjust padding
* Change the colors to the font and background

== Installation ==

1. Unpack the 'simple-copyright-footer.zip' file to the 'wp-content/plugins/' directory inside of WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress
 
== Frequently Asked Questions ==
 
= What is the main purpose of this plugin? =
 
Use this plugin to insert any copyright and/or all rights reserve text to your footer. This text will also appear when using the copyright shortcode.
 
= What edits can I make with this plugin? =
 
You can edit the text, display, text alignment, padding, font size, font color, and bar color.

= Where is the plugin located? =

Once installed, you can find this plugin under `Dashboard > Tools > Copyright`.

= Can I purpose changes to this plugin? =
 
Yes, and please do. You can visit our website to contact us and we will gladly respond to your request.
